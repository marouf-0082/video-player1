فایل videoControls.js

const control_container = document.querySelector('.controls-main');
این خط عنصر HTML که دارای کلاس .controls-main است را انتخاب می‌کند و در متغیر control_container ذخیره می‌کند.

const video = document.querySelector('video');
این خط تگ video را در صفحه انتخاب می‌کند و در متغیر video ذخیره می‌کند.

const play_btn = document.querySelector('.btn-play i');
این خط آیکون داخل دکمه پخش را که درون کلاس .btn-play قرار دارد انتخاب می‌کند.

function play_video() { ... }
این تابع بررسی می‌کند که آیا ویدیو در حال پخش است یا متوقف شده است. اگر متوقف باشد، ویدیو را پخش می‌کند و آیکون دکمه را تغییر می‌دهد. در غیر این صورت، ویدیو را متوقف می‌کند و آیکون را به حالت fa-play تغییر می‌دهد.

video.addEventListener("timeupdate", () => { ... });
این خط باعث می‌شود که هر زمان که موقعیت زمانی ویدیو تغییر کند، مقدار زمان جاری در نمایشگر به‌روز شود.

فایل keyboardControls.js

window.document.onkeydown = (e) => { ... }
این خط یک شنونده رویداد برای کل صفحه تعریف می‌کند که وقتی کلیدی روی صفحه‌کلید فشرده شد، بررسی کند که چه کلیدی فشرده شده است.

case 'ArrowUp': // volume increase
اگر کلید ArrowUp فشار داده شود، میزان صدای ویدیو افزایش پیدا می‌کند (تا حداکثر مقدار 1).

case 'ArrowDown': // sound reduction
اگر کلید ArrowDown فشار داده شود، میزان صدای ویدیو کاهش پیدا می‌کند (حداقل مقدار 0).

case ' ': // play or pause
اگر کلید اسپیس (فاصله) فشار داده شود، ویدیو پخش یا متوقف می‌شود.

case 'ArrowRight': // fast forward video
اگر کلید ArrowRight فشار داده شود، ویدیو 5 ثانیه به جلو می‌رود.

case 'ArrowLeft': // fast backward video
اگر کلید ArrowLeft فشار داده شود، ویدیو 5 ثانیه به عقب می‌رود.

case 'f': case 'F': // fullscreen
اگر کلید F یا f فشار داده شود، ویدیو به حالت تمام صفحه تغییر پیدا می‌کند.

case 'm': case 'M': // mute
اگر کلید M یا m فشار داده شود، صدای ویدیو قطع یا وصل می‌شود.
